# ラピス

ラピス（Lapis）は、デカダンス住民にとって不可欠なスマート端末であり、通信・決済・魔導操作を一体化した多機能デバイス。

手首や耳元、胸部装甲に埋め込むタイプが主流で、使用者の意識や声に反応してホログラム画面や魔法回路を展開する。

個人認証と生体リンクにより所有者固有の魔力に最適化され、ミラージュジェムの武装展開やF.PheroSyncとのフェロモン連携も可能。

ラピスは記憶や感情のログを保管し、仲間との絆を深める“心の鍵”としての役割も持つ。

機能は都市の階層や職業により差があり、上級モデルほど高性能な共鳴演算を搭載。

中には人格を持つ特注モデルも存在し、持ち主と会話するものもある。
