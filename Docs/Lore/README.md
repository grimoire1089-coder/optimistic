# Lore

このフォルダは、ゲーム内の伝承・世界観・キャラクター設定・土地や組織の設定を置く場所です。

Codex がゲーム内容に関わる実装や文章調整を行う時は、必要に応じてこのフォルダ内の資料を参照します。

## 置くとよいもの

- 世界観の基本設定
- 主要キャラクターの設定
- 場所や施設の設定
- 組織・企業・勢力の設定
- アイテムや料理の由来
- 年表や過去の出来事
- 用語集

## 運用メモ

- 設定が確定しているものと、まだ仮のものは分けて書く。
- 実装に影響する設定は、できるだけ具体的に書く。
- コードやデータに反映した設定を変える時は、関連ファイルも確認する。
