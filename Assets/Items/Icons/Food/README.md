# Food icon assets

食品・料理・食材の画像はこのフォルダに置く。

Godot path:

```gdscript
res://Assets/Items/Icons/Food/
```

## 用途

- 食材アイコン
- 料理アイコン
- インベントリやレシピUIで使う食品画像

## 命名ルール

- 小文字の snake_case を使う
- 拡張子は基本 `.png`
- 例:
  - `nikuman_basic.png`
  - `toast_plain.png`
  - `avocado_cheese_harumaki.png`

## 置き分け

- UIボタンやUI専用アイコンは `res://Assets/UI/Icons/`
- 食品・食材・料理の画像は `res://Assets/Items/Icons/Food/`
