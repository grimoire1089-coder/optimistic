# Furniture Image Assets

家具画像素材の格納場所です。

このプロジェクトでは、家具画像はマップ上のグリッド配置に合わせて管理します。

## 基本ルール

- 1グリッド = 48px
- 家具画像はグリッド占有サイズが分かる名前にします
- 手描き風・ソフト陰影の家具は、基本的に2倍解像度で保存します
- Godot側で必要に応じて `Sprite2D.scale = Vector2(0.5, 0.5)` にして表示します

## 推奨解像度

| 家具サイズ | ゲーム内表示 | 推奨PNG原寸 |
|---|---:|---:|
| 1x1 | 48x48 | 96x96 |
| 2x1 | 96x48 | 192x96 |
| 2x2 | 96x96 | 192x192 |
| 4x2 | 192x96 | 384x192 |

## 命名ルール

```text
<item_id>_<width>x<height>.png
```

例:

```text
simple_mattress_4x2.png
```

2倍原寸であることを明示したい場合:

```text
simple_mattress_4x2@2x.png
```

ただし、ゲーム内で使うファイルを1つに絞る場合は `@2x` を付けず、ファイルの中身を2倍解像度にしておきます。

## カテゴリ

- `Bedroom/` ベッド、マットレス、棚、照明など
- `Kitchen/` キッチン、冷蔵庫、調理台など
- `Storage/` チェスト、箱、収納など
- `Decor/` 装飾品、ラグ、観葉植物など
- `Utility/` 作業台、端末、機械設備など
